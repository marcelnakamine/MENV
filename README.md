# MEVN
**M**ongoDB **E**xpress **V**ue **N**ode

## Client

```
$ cd client
$ npm install
$ npm run dev
```

## Server

### Start Mongo DB

### Start server

```
$ cd server
$ npm install
$ npm start
```

### For windows
```
$ cd server
$ npm install
$ npm install -g nodemon
$ nodemon src/app.js
```
